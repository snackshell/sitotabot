# Security Policy

## Supported Versions

Currently, only the latest release of SitotaBot is supported with security updates.

## Reporting a Vulnerability

We take the security of SitotaBot seriously. If you discover a security vulnerability, please **DO NOT** open a public issue.

Instead, please send an email to the project maintainers (or message them securely) with the details of the vulnerability. We will endeavor to respond within 48 hours and work with you to patch the issue as quickly as possible.

### What to include in your report
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any suggestions for remediation.

Thank you for helping keep SitotaBot secure!
